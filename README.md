# 🧠 Personal AI Assistant — Powered by Gemini

Your all-in-one AI assistant for college life — task management, day planning, emails, and more.

## ✨ Features
- **Task Manager** — Prioritize & schedule your to-do list
- **Day Planner** — Hour-by-hour optimal schedule
- **Email Writer** — Professional emails for placements & college
- **Content Reviewer** — Feedback on essays, SOPs, cover letters
- **Idea Generator** — Project, startup & hackathon ideas

## 🚀 Deploy to Streamlit Cloud
1. Push this folder to a GitHub repo
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect repo → select `app.py` → Deploy!

## 📦 Local Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
